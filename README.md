Features 

- React Navigation (Stack)
- Trip Management (view, add, delete trips as well as view trip details and participants)
- State Management using Zustand to manage trips and participants

How to build your own?

1. Clone this repo or download the zip
   - Extract Zip file
   - Open folder in IDE
   
3. Install all the dependencies
   - npm i or npm install
     
4. Run any of the following commands to start the application
   -npm run android
   -npm expo start
   -npm start -- --reset-cache

5. Test the different functions of the App and tweak code to obeserve new changes.
